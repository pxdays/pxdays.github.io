01 About

I'm Joshua, 15, from the UK. I build CLI tools — small, focused utilities that live in the terminal and do one thing well.

Every tool is built with DeepSeek V4 Flash, tested for syntax errors, packaged with full source code, and published to my store. No DRM. No accounts. No subscriptions. Pay once in Litecoin, own it forever.

I'm self-taught, I use Arch Linux (btw), and I believe good tools don't need a GUI. 40+ tools built so far.
